Frequently Asked Questions

How do I improve the mobile network with this app?

The NETworks project aims to improve existing network coverage maps by identifying discrepancies and issues that affect real-life mobile network connections. By engaging everyday users, we aim to analyze these maps more effectively. Our goal is to pinpoint areas needing network improvements and ensure that coverage maps align better with actual user experiences. Ultimately, we aim to identify regions that require enhanced network quality.

What is the difference between 5G, 4G, H+ ...?

The abbreviations represent the different generations of mobile network technology.
  • 2G: Second generation, primarily for voice calls and limited data transfer.
  • 3G: Stands for third generation, providing faster data transfer speeds than 2G.
  • H+: An enhancement of 3G, offering improved speeds before the widespread adoption of 4G.
  • 4G (LTE): Provides high-speed internet for mobile devices, offering faster speeds than its predecessors.
  • 5G: The latest generation, offering ultra-fast speeds, low latency, and high capacity for data transmission.
Each generation represents advancements in speed, capacity, and technological capabilities, with newer generations typically offering faster data transfer and lower latency compared to their predecessors.

How is the mobile connection tied to the network speed?

The mobile connection speed determines how quickly data travels between your device and the network. Factors like network generation (2G, 3G, 4G, 5G), signal strength, network congestion, location, and device capability influence this speed, affecting how fast you can browse, stream, or download on your mobile device.

Which external factors affect mobile network connection?

Several external factors can affect mobile network connections:
  • Physical Obstructions: Buildings, terrain, and natural features can block or weaken signals, causing signal loss or reduced quality.
  • Distance from Towers: Proximity to cell towers affects signal strength. Being too far might result in weaker signals and slower connections.
  • Weather Conditions: Extreme weather (rain, snow, storms) can interfere with signals, causing disruptions or slower speeds.
  • Network Congestion: High usage in an area can lead to congestion, slowing down data speeds during peak times.
  • Interference: Electronic devices, metal structures, or electromagnetic interference can disrupt signals.
  • Roaming: Using a different carrier's network can result in varying signal quality and connection speeds.
These external factors collectively influence the reliability, speed, and quality of your mobile network connection.

I can not see my data trajectories on the map. Why?

There are a few reasons why this can happen.
  • If you did not track any data yet, there will also be no data displayed. Check the "total km tracked data" in the "Personal Data Analysis" tab.
  • Make sure to always login in with the same user ID. Only then the data from all the login sessions will be displayed.
  • The default settings will adjust the window to your current location. Zoom in and out in different regions on the map. Your trajectories might not be visible in the current window but appear when changing it.
If your data still is not visible please contact us.

What is so special about trains considering mobile network connections?

Signal repeaters on trains amplify and distribute mobile signals within carriages, improving connectivity for passengers. They capture signals from cell towers, reducing disruptions caused by speed, tunnels, or remote areas. These repeaters alleviate congestion, enhancing connections and data speeds. Despite their benefits, their effectiveness depends on initial signal quality, repeater design, and geographical challenges.

For the NETworks project train amplifiers falsify the information about the mobile connection around train tracks. Therefore tracked data along train tracks is filtered and not taken into account when analysing the data.

How does the reward program work?

When you track data, the length of the tracked trip is calculated. The longer your trip, the greater the benefit to our project. The length of all your trips with the same user ID is added and for every 10km you get 0.1 CHF discount on your current payment of mobile plan. The total discount is displayed in the "personal data analysis" tab. Every month the discount and the total km tracked will be reset.

Your discount code will only work if you have a monthly mobile subscription with one of the three Swiss network providers: Salt, Sunrise or Swisscom. Each discount code is for single use only. Go to your network provider's website to enter the code.

Which providers support the app?

The three Swiss network providers: Salt, Surnise, Swisscom

For how long is my discount valid?

Your discount accumulates throughout the month. On the first day of the new month, the discount is reset and the code changes. You can enter your code once to receive a discount on your current monthly payment for your mobile plan. The code will only work on a single payment.
